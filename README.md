# I Have No Insomnia

Mod that allows players to individually disable phantom spawns. Gamerule `doInsomnia` still enables/diables them globaly

<img width="748" height="421" alt="NoPhantoms" src="https://github.com/user-attachments/assets/e2b26989-b717-4668-968f-7202d915730c" />

`/insomnia enable` - enables phantoms spawn

`/insomnia disable` - disables phantoms spawn
